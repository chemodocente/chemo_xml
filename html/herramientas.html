Herramientas para crear una Web.

Herramientas necesarias

  • Navegador
  • Editor de texto
  • Carpeta de proyecto
  • Imagenes
  • Textos

Pasos para crear la primera pagina

  1. Abrimos el Editor de texto
  2. Creamos un archivo nuevo con extension xml
  3. Montamos la estructura base de html (truco !)
  4. Cambiamos los datos necesarios en title
  5. Y maquetamos con nuestras etiquetas de HTML H1-H6 y P para hacer titulos y parrafos

Debemos seguir los pasos en orden para que no nos salter errores durante el desarrollo y poder ir viendo como queda de manera correcta.
